My Works

This is the first background I created. We required a simple repeating backround to fit with the theme of the website. I grabbed the boomerang from the main wallaru art and repeated it in a slightly different position throughout the image. This adds more flow to the background.

For this task, I was required to create three different sizes of the main art for the Wallaru site: desktop, tablet, and mobile. I had to scale down the images in order to not lose quality in the image and retain the appropriate width and height for those different screen sizes.

In the News page of the Wallaru site, we wanted a simple repeating background and needed the basic layout for the page. I implemented both and created the basic layout for how we wanted the news items to look, which was three across in a row and repeat down the page.

For the mobile view of the Wallaru page, we needed to make it responsive so that it is easy for users to view the page on mobile devices. My task was to create the hamburger icon and hide the main navigation bar items in mobile view, so instead they show up in the mobile menu when it is clicked.

I was tasked to change the Pirate101 splash page and implement the new Pirate101 animated background as we wanted to make the page more eye-catching. I assembled the layout and implemented the animated background, headline text, logo, Play Now button, and the other Pirate101 page links.

As part of the above project for Pirate101, we wanted the background for mobile view to be a static image. I implemented the maximum width of the screen, so if it falls below the threshold (if it is being viewed on a mobile device), the background will switch to a static background. I also changed the layout on mobile view so that all of the elements on the page were more in a column layout.
